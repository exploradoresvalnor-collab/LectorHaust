# LectorHaus 📖✨

Tu puerta al mundo del manga, manhwa y manhua con una experiencia premium y profesional.

## Características

- 🚀 Interfaz ultrarrápida impulsada por Vite e Ionic.
- 🎨 Diseño moderno con glassmorphism y animaciones fluidas.
- 🔍 Filtros de búsqueda avanzados y profesionales.
- 📚 Biblioteca persistente para seguimiento de lecturas.
- 🌓 Modo oscuro nativo.
- 📱 Optimización total para dispositivos móviles y escritorio.

## Desarrollo

```bash
npm install
npm run dev
```

Desarrollado con ❤️ para los amantes del manga.
